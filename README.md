# ESP32 Bluetooth Reversing

This repository contains an effort to reverse engineer the ESP32 Bluetooth peripheral and document it in order to move towards an open Bluetooth controller implementation.

## Contents

| Folder                                                         | Description                                                                                          |
| -------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| [esp32-bluetooth-golden-binary](esp32-bluetooth-golden-binary) | ESP-IDF project designed to generate an ELF file with all Bluetooth related functions for the ESP32. |
| [documentation](documentation)                                 | Bluetooth core peripheral documentation.                                                             |
